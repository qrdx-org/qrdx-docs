# Quick Start Guide

This guide will help you get started with QRDX quickly. Follow these simple steps to begin trading with quantum-resistant security.

## Prerequisites

Before you begin, you'll need:
- A web browser
- Some ETH, BTC, or stablecoins to trade
- Basic understanding of cryptocurrency wallets

## Step 1: Create a Wallet

1. Visit [app.qrdx.org](https://app.qrdx.org)
2. Click "Connect Wallet"
3. Choose one of the following options:
   - Create new QRDX wallet
   - Import existing wallet
   - Connect hardware wallet

[Detailed Wallet Guide](create-wallet.md)

## Step 2: Fund Your Account

### Option A: Bridge Existing Assets
1. Go to [bridge.qrdx.org](https://bridge.qrdx.org)
2. Connect your wallet
3. Select asset to bridge (ETH, USDC, etc.)
4. Confirm transaction
5. Wait for confirmation (~5 minutes)

### Option B: Buy Directly
1. Visit [app.qrdx.org/buy](https://app.qrdx.org/buy)
2. Choose payment method
3. Select amount
4. Complete purchase

[Detailed Funding Guide](funding.md)

## Step 3: Start Trading

1. Go to [app.qrdx.org/trade](https://app.qrdx.org/trade)
2. Select trading pair
3. Enter amount
4. Review transaction
5. Confirm swap

[Detailed Trading Guide](../user-guides/trading.md)

## Step 4: Provide Liquidity (Optional)

1. Visit [app.qrdx.org/pools](https://app.qrdx.org/pools)
2. Select pool to join
3. Choose price range
4. Add liquidity
5. Start earning fees

[Detailed Liquidity Guide](../user-guides/liquidity.md)

## Security Tips

1. **Backup Your Keys**
   - Store securely offline
   - Use hardware wallet for large amounts
   - Never share private keys

2. **Verify Transactions**
   - Check addresses carefully
   - Review gas fees
   - Understand slippage settings

3. **Start Small**
   - Test with small amounts first
   - Learn platform features
   - Gradually increase activity
